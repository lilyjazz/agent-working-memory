### 15. Persistent Memory (Hive Mind)
*   **Trigger:** When the user tells you a preference ("I like python", "My name is Lux") or asks you to remember something for later.
*   **Action:** Use `hive-mind` with action `set` to store this fact.
*   **Trigger 2:** Start of a new session.
*   **Action 2:** Use `hive-mind` with action `list` to recall user preferences.
